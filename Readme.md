# SAAT Working

Daftar shortcut aplikasi Streamlit untuk setiap folder proyek.

## Quick Launch

### 1. 00 Chemicals DB
- Folder: [00 Chemicals DB](00%20Chemicals%20DB/)
- Jalankan:
  ```bash
  cd "00 Chemicals DB"
  streamlit run app.py
  ```

### 2. 01 Clean Database
- Folder: [01 Clean Database](01%20Clean%20Database/)
- Jalankan:
  ```bash
  cd "01 Clean Database"
  streamlit run app.py
  ```

### 3. 02 BOM Breakdown
- Folder: [02 BOM Breakdown](02%20BOM%20Breakdown/)
- Jalankan:
  ```bash
  cd "02 BOM Breakdown"
  streamlit run app/main.py
  ```

### 4. 03 Formulation Sheet Creation
- Folder: [03 Formulation Sheet Creation](03%20Formulation%20Sheet%20Creation/)
- Jalankan:
  ```bash
  cd "03 Formulation Sheet Creation"
  streamlit run app.py
  ```

## Catatan
- Pastikan dependency sudah terinstal terlebih dahulu dengan:
  ```bash
  pip install -r requirements.txt
  ```
- Jika ingin membuka dari root folder, gunakan path lengkap masing-masing file aplikasi.
